📖 Описание

«Оператор Архитектор» — это интерактивный 3D-симулятор управления беспилотным летательным аппаратом (БПЛА), разработанный в рамках 

образовательной платформы Anti-AI Qwen Academy.

Игрок управляет детальным дроном, который должен провести инспекцию 4-этажного здания, облетая его со всех сторон и поражая мишени, 

расположенные в окнах. Тренажёр развивает навыки точного пилотирования, пространственного мышления и реакции.

⚠️ Главное правило: Столкновение со стеной здания или падение на землю = немедленный провал миссии!
